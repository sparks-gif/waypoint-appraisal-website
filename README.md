# Waypoint Appraisal Website

Professional website for Waypoint Appraisal Co. - Real estate appraisal services in North Mississippi.

## Build & Deploy

- **Build command:** `npx vite build --config vite.config.cloudflare.ts`
- **Output directory:** `dist/public`

## Development

```bash
npm install
npm run dev
```
